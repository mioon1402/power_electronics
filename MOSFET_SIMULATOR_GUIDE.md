# 🔌 MOSFET 시뮬레이터 - 티스토리 사용 가이드

## 📋 개요

이 MOSFET 시뮬레이터는 **HTML 한 파일**로 자체 포함되어 있어 티스토리 블로그에 바로 붙여넣을 수 있습니다.

## 🚀 빠른 시작

### 1단계: 파일 다운로드
`mosfet_simulator.html` 파일을 다운로드합니다.

### 2단계: 티스토리에 업로드
1. 티스토리 블로그의 **글 쓰기** 페이지 열기
2. **HTML/JavaScript 편집기** 모드로 전환
3. `mosfet_simulator.html` 파일을 열어서 **전체 코드 복사**
4. 티스토리 편집기에 **붙여넣기**

### 3단계: 확인 및 발행
- 미리보기에서 시뮬레이터가 제대로 작동하는지 확인
- 글 발행

## ⚙️ 주요 기능

### 1. **실시간 수정 가능한 슬라이더**
- **Vgs (게이트-소스 전압)**: 0~10V 조절
- **Vds (드레인-소스 전압)**: 0~50V 조절

### 2. **자동 계산 파라미터**
- **Id (드레인 전류)**: mA 단위로 표시
- **P (전력소비)**: mW 단위로 계산
- **Ron (에이온 저항)**: Ohm 단위
- **gm (트랜스컨덕턴스)**: mS 단위

### 3. **작동 영역 자동 표시**
- 🔴 **차단 영역 (Cutoff)**: Vgs < Vth
- 🔵 **선형 영역 (Linear)**: Vgs > Vth, Vds < Vgs - Vth
- 🟢 **포화 영역 (Saturation)**: Vgs > Vth, Vds > Vgs - Vth

### 4. **I-V 특성 곡선**
- 여러 Vgs 값에 대한 특성 곡선을 동시에 표시
- 빨간 점으로 현재 작동점 표시
- interactive 그래프로 마우스로 값 확인 가능

## 📊 MOSFET 모델 파라미터

기본 설정된 NMOS 모델:
```
Vth (임계전압): 1.5V
Kn (트랜스컨덕턴스): 0.05
Lambda (채널 길이 변조): 0.02
```

### 파라미터 수정 방법
HTML 코드의 다음 부분을 찾아서 수정:

```javascript
const mosfetParams = {
    Vth: 1.5,           // 임계전압
    Kn: 0.05,           // 트랜스컨덕턴스 파라미터
    lambda: 0.02        // 채널 길이 변조 파라미터
};
```

## 💡 교육 활용

이 시뮬레이터는 다음 학습 주제에 유용합니다:
- MOSFET의 I-V 특성 이해
- 에이온(On-resistance, Ron) 개념 학습
- 트랜스컨덕턴스(Transconductance) 이해
- 작동 영역(Cutoff, Linear, Saturation) 구분

## ❓ 자주 묻는 질문

**Q: 다른 MOSFET 모델을 시뮬레이션할 수 있나요?**
A: 네, `mosfetParams`의 Vth, Kn, lambda 값을 변경하면 다른 MOSFET 모델을 시뮬레이션할 수 있습니다.

**Q: 티스토리 외에 다른 플랫폼에서도 사용 가능한가요?**
A: 네, 모든 HTML 기반 블로그 플랫폼(GitHub Pages, Notion, Wix 등)에서 사용 가능합니다.

**Q: 오프라인에서도 사용할 수 있나요?**
A: 아니요, Chart.js 라이브러리를 CDN에서 로드하므로 인터넷 연결이 필요합니다.

## 📞 문제 해결

**시뮬레이터가 표시되지 않는 경우:**
1. 인터넷 연결 확인 (CDN 라이브러리 로드 필요)
2. 브라우저 개발자 도구(F12)에서 콘솔 에러 확인
3. HTML 코드가 완전히 복사되었는지 확인

**그래프가 나타나지 않는 경우:**
1. 페이지 새로고침(Ctrl+R)
2. 다른 브라우저 시도
3. 브라우저 캐시 삭제
